    Беспроводной пульт управленния машинки 
    Использованные материалы:
      Arduino UNO R3;
      NRF24L01;
      Joystick обычный;
      Powerbank;
      Распиновка:
          --------------------------------
          NRF24L01:
          CE - D10, 
          CSN - D9, 
          VCC - 3.3, 
          GND - GND, 
          MOSI - D11, 
          MISO - D12, 
          SCK - D13
          --------------------------------
          Джойстик для управления:
          X: A0;
          Y: A1;
          Y2:A3;
          --------------------------------
          Питание: Powerbank
          --------------------------------
